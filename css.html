CSS (Cascading Style Sheets) is a style sheet language used to style and describe the presentation of a document written .

A CSS selector selects the HTML element you want to style.

Categories of Css selectors are;

-Attribute selectors

-Simple selectors

-Combinator selectors

-Pseudo selectors

Css can be applied internally or externally

Internal Css is done inside the html file Whereas External Css is done in an independent Css file where the Css file is linked up with the Html file

The style attribute is usually used in internal Css

In external Css the following selectors are used

-Unviversal selector(*)

e.g.

*{

padding: 0;

margin: 0;

list-style: none;

box-sizing: border-box;

text-decoration: none;

}

-element selector( the element itself is used in the Css

e.g.

h2 {

text-align: center;

color: red;

}

-class selector(.)

e.g.

.today{

background-image: url('images/Vase.png');

}

-id selector(#)

e.g.

#ten{

font-size: 20px;

}
